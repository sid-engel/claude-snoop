# 🐾 claude-snoop

> ⚠️ **Super early alpha. Here be dragons.**

AI-powered network audit tool built on [Claude Code](https://claude.ai/code). Spins up specialized sub-agents to scan and audit a network, then synthesizes the findings into a readable report.

This thing is in early development. Architecture, CLI, and output format are all subject to change. If you're here, you're early.

---

## The Idea

Point it at a network. Walk away with a report.

Sub-agents handle specific audit tasks — discovery, ports, vulnerabilities, SMB, AD, M365, etc. An orchestrator ties it together and a report generator turns the findings into something you can actually hand to someone.
